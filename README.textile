h1. Qt::JRuby DSL

h2. Description

Give a little sugar to Qt::JRuby.

h2. Installation

Install "Qt::JRuby Core":http://github.com/nmerouze/qtjruby-core. Then :

<pre><code>git clone git://github.com/nmerouze/qtjruby-dsl.git
cd /path/to/qtjruby-dsl
jruby -S rake install</code></pre>

h2. Example

<pre><code>require 'qtjruby-dsl'

Cute.app do
	button('Click Me!') { alert 'Clicked!' }
end</code></pre>
